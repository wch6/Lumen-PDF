# Font manifest

This folder centralizes optional font source files kept with the app assets.
They are not declared in `pubspec.yaml` as Flutter assets or app fonts, so they
are not bundled into the compiled application unless a future feature explicitly
references them.

## Downloaded from Noto official repositories

- `NotoSans-Regular.ttf`
  - Source: `https://raw.githubusercontent.com/notofonts/noto-fonts/main/hinted/ttf/NotoSans/NotoSans-Regular.ttf`
- `NotoSansCJKsc-Regular.otf`
  - Source: `https://raw.githubusercontent.com/notofonts/noto-cjk/main/Sans/OTF/SimplifiedChinese/NotoSansCJKsc-Regular.otf`
