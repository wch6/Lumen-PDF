# Lumen PDF Windows Installer

This directory contains the reproducible Windows installer pipeline.

The installer is built with Inno Setup 6. It creates a standard Windows setup wizard where users can choose the installation directory, choose whether to create a desktop shortcut, install Lumen PDF, and later uninstall it from Windows Settings or the Start menu.

## Requirements

Install Inno Setup 6. The current local compiler path is:

```text
D:\Inno_Setup_6\ISCC.exe
```

The build script also tries `ISCC.exe` from PATH, registry install locations, and common Inno Setup folders.

## Build

From the repository root:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\installer\build_installer.ps1
```

The output is written to:

```text
build\installer\dist\LumenPDF-Setup-1.0.0.exe
```

Use the pubspec version by default, or pass a version explicitly:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\installer\build_installer.ps1 -BuildName 1.0.1
```

Skip the Flutter build when the release bundle is already current:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\installer\build_installer.ps1 -SkipFlutterBuild
```

Use a custom Inno Setup compiler path:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\installer\build_installer.ps1 -InnoSetupCompiler D:\Inno_Setup_6\ISCC.exe
```

Build without the desktop shortcut task:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\installer\build_installer.ps1 -NoDesktopShortcut
```

## Installer Behavior

- Default install directory: `{autopf}\Lumen PDF`.
- The directory page is always shown, so users can choose another folder.
- The installer runs with current-user privileges by default.
- The Start menu shortcut is created under `Lumen PDF`.
- The desktop shortcut is optional in the wizard.
- The app can be launched at the end of setup.

## Uninstall Behavior

During interactive uninstall, the uninstaller asks whether to delete all user data and cache generated by Lumen PDF.

If the user confirms, it removes:

- `%LOCALAPPDATA%\LumenPDF`
- Legacy local data folders from earlier builds
- Legacy roaming data folders from earlier builds

This includes settings, recent files, reading positions, notes, highlights, and local cache databases. Original PDF files are not deleted.

Silent uninstall does not delete user data.

## Notes

- The generated installer is unsigned. Windows SmartScreen may warn users until the installer is code-signed.
- For public distribution, sign both `lumen.exe` and the generated setup executable.
